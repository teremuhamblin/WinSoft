# 📚 Documentation — WinSoft

Ce dossier contient toute la documentation officielle du projet **WinSoft**.  
Il est structuré pour être compatible avec une future activation de **GitHub Pages**, sans dépendance externe.

## 📁 Contenu

- `index.md` — Page d’accueil de la documentation  
- `install.md` — Guide d’installation et d’exécution  
- `usage.md` — Utilisation du script principal et des modules  
- `modules.md` — Description détaillée des modules (network, performance, cleanup)  
- `config.md` — Structure et fonctionnement des fichiers de configuration  
- `troubleshooting.md` — Résolution des problèmes courants  

## 🎯 Objectifs

- Fournir une documentation claire et centralisée  
- Préparer la mise en ligne via GitHub Pages (non activé pour l’instant)  
- Faciliter la contribution et la maintenance du projet  

## 🔧 Notes

- Toute nouvelle fonctionnalité doit être documentée ici.  
- Les fichiers doivent rester en **Markdown simple** pour compatibilité maximale.
