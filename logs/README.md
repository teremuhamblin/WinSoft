# 📁 Dossier : logs/

Ce dossier contient les fichiers journaux générés automatiquement par les scripts de **WinSoft**.

## Contenu

- Fichiers `.log` créés à chaque exécution
- Historique des actions effectuées
- Informations utiles pour le diagnostic ou le suivi

## Rôle

Les logs permettent de vérifier :
- les modules exécutés
- les actions réalisées
- les éventuelles erreurs rencontrées

Ils sont générés automatiquement par `main.ps1`.
