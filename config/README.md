# 📁 Dossier : config/

Ce dossier contient les fichiers de configuration utilisés par le projet **WinSoft**.

## Contenu

- **apps-common.txt**  
  Liste des applications à supprimer sur Windows 10 et Windows 11.

- **apps-win10.txt**  
  Liste des applications spécifiques à Windows 10 à désinstaller.

- **apps-win11.txt**  
  Liste des applications spécifiques à Windows 11 à désinstaller.

## Rôle

Ces fichiers permettent d’adapter facilement le comportement des scripts sans modifier le code PowerShell.
