# 🖼️ Assets — WinSoft

Ce dossier contient toutes les ressources visuelles utilisées par le projet **WinSoft**.

## 📁 Contenu

- `banner.png` — Bannière principale du projet  
- `logo.svg` — Logo officiel WinSoft  
- `screenshots/` — Captures d’écran du script, modules et interface console  

## 🎯 Objectifs

- Centraliser les éléments graphiques du projet  
- Faciliter l’intégration dans la documentation et le README principal  
- Préparer les visuels pour une future publication GitHub Pages  

## 📝 Bonnes pratiques

- Utiliser des formats légers (PNG, SVG, WebP)  
- Nommer les fichiers de manière explicite  
- Placer les captures d’écran dans `assets/screenshots/`  
- Ne jamais stocker de fichiers volumineux ou inutiles ici
