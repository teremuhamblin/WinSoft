# 📁 Dossier : win11/

Ce dossier contient les modules spécifiques à **Windows 11** pour le projet **WinSoft**.

## Contenu

- **debloat-win11.ps1**  
  Suppression des applications propres à Windows 11 (Widgets, Teams Chat, Clipchamp, etc.).

- **optimize-win11.ps1**  
  Optimisations adaptées aux particularités de Windows 11.

## Rôle

Ces scripts ne sont exécutés que si le système détecté est Windows 11.
