# 📁 Dossier : common/

Ce dossier contient les scripts communs à Windows 10 et Windows 11 pour le projet **WinSoft**.

## Contenu

- **debloat-common.ps1**  
  Suppression des applications présentes sur les deux systèmes.

- **optimize-common.ps1**  
  Optimisations générales valables pour Windows 10 et Windows 11.

## Rôle

Ces scripts sont exécutés avant les modules spécifiques à chaque version de Windows.
