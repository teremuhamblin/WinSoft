# 📁 Dossier : win10/

Ce dossier contient les modules spécifiques à **Windows 10** pour le projet **WinSoft**.

## Contenu

- **debloat-win10.ps1**  
  Suppression des applications propres à Windows 10.

- **optimize-win10.ps1**  
  Optimisations ciblées pour améliorer la réactivité de Windows 10.

## Rôle

Ces scripts ne sont exécutés que si le système détecté est Windows 10.
